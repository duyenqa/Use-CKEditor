# Use-CKEditor

## Version 4
> 1. Get the content from the editor

```Javascript
CKEDITOR.instances.richtext.getData();
```

> 2. Set the content from the editor

```Javascript
CKEDITOR.instances.richtext.setData() = " ";
```

## Version 5

> 1. Get the content from the editor

```Javascript
window.editor.getData();
```

> 2. Set the content from the editor

```Javascript
window.editor.setData(" ");
```
